# IFA MAKER

![Logo](./ToDoListApp.png)

Repository de mon projet fil-rouge 'IFA Maker' dans le cadre de ma formation en développement web.
Il s'agit d'un site de gestion de projets en vue Kanban, similaire à Trello.com.

*Repository of my main school project at my web-developping academy*
*It's a project management kanban tool, like Trello.com*

## Technologies

 - HTML (Bootstrap)
 - CSS
 - PHP
 - Javascript (Jquery, Jquery UI, AJAX)
 - MySQL

## Utilisation / How to use it :

Vu que ce projet est réalisé principalement en PHP, vous devez disposer d'un serveur web ou d'un environnement permettant de le simuler en local (type WAMP, MAMP) pour lancer le site.

*Due to the using of PHP for most of the website, in order to use it you'll need a web server to run it or a solution like WAMP or MAMP to simulate a local server.*
